# Source des données

Ici nous recensons les sources de données utilisées pour le projet, classées par indicateurs.

## Environnement

- arbre (total) : https://opendata.paris.fr/explore/dataset/les-arbres/information
- espaces vert : https://opendata.paris.fr/explore/dataset/espaces_verts/information
- ilots de fraicheurs : https://opendata.paris.fr/explore/dataset/ilots-de-fraicheur-equipements-activites/information
- déchets trilib : https://opendata.paris.fr/explore/dataset/dechets-menagers-points-dapport-volontaire-stations-trilib/information

## Mobilité

- vélib : nombre station + nombre vélo https://opendata.paris.fr/explore/dataset/velib-emplacement-des-stations/information
- métro : https://www.data.gouv.fr/datasets/lignes-et-stations-de-metro-en-france/
- bus : https://prim.iledefrance-mobilites.fr/fr/jeux-de-donnees/arrets-lignes?tab=carte
